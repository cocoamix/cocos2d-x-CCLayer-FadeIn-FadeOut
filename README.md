cocos2d-x-CCLayer-FadeIn-FadeOut
================================


Usage
------
1) Place the parts(CCSprite, CCLabelTTF,,) in this layer .  
2) Perform runAction .  
e.g.  
Layer->runAction(CCFadeIn::create(2.0));  

3) runAction will affect all .
